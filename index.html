Panduan Setup IRL Stream

Langkah 1: Setup Jaringan (Wajib)

  • Download aplikasi Tailscale di HP dan Laptop.
  • Aktifkan Tailscale di HP. Pastikan VPN IP terbaca.
--- PILIH METODE KAMERA ---

Pilihan A: Kamera Eksternal (USB Camera Pro)

  • Hubungkan Kamera Fisik ke HP via OTG. Buka USB Camera Pro.
  • Pengaturan > Stream Settings. Type: RTMP/RTSP Server.
  • Masukkan URL dan Stream ID. Tekan START STREAM.

Pilihan B: Kamera Bawaan HP (IRL Pro)

  • Buka IRL Pro. Settings > Video, Bitrate 2500-4000 kbps, Codec HEVC.
  • Menu Connections. Masukkan URL dan Stream ID. Tekan LIVE.
--- KONTROL & OBS STUDIO ---

Fitur Dashboard Kontrol

  • Core Engine: Pastikan selalu menyala (tombol merah/hijau).
  • Start/Stop Preview: Pantau video dan grafik jaringan.
  • Sensor Otomatis: Otomatis memutus preview jika HP offline.

Langkah Akhir: Menarik Video ke OBS

  • Di OBS Studio, tambahkan Media Source, matikan Local File.
  • Paste link Media Source (SRT). Ketik mpegts di Input Format.
