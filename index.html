📖 BUKU PANDUAN IRL SYSTEM V1

1️⃣ Persiapan Jaringan (Tailscale)

  • Pastikan aplikasi Tailscale sudah terinstal dan dalam status Active/Connected di HP dan Laptop Anda.
  • Cek kolom VPN IP (Tailscale) di aplikasi ini. Pastikan angkanya berawalan 100.x.x.x. Jika tulisannya "Tailscale Mati", periksa kembali koneksi VPN di laptop Anda.

2️⃣ Menyalakan Server (Core Engine)

  • Klik tombol 🟢 Start Core Engine pada panel kiri.
  • Tombol akan berubah menjadi merah (🔴 Stop Core Engine) dan status jaringan di bawah layar akan berubah menjadi • STANDBY.
  • Ini artinya server PC Anda sudah siap menerima pancaran video dari lapangan.

3️⃣ Setting Aplikasi Kamera di HP

Pilih salah satu aplikasi di bawah ini sesuai perangkat Anda:

  • Bagi Pengguna IRL Pro (Kamera Bawaan HP):
    • Buka menu Settings > Video. Atur Bitrate ke 2500 - 4500 kbps, and Video Codec WAJIB menggunakan HEVC / H.265.
    • Masuk ke menu Connections, tambahkan koneksi baru jenis SRT.
    • Paste SRT Server URL dan SRT Stream ID dari aplikasi ini ke kolom yang tersedia. Tekan tombol LIVE.
  • Bagi Pengguna USB Camera Pro (Kamera Eksternal):
    • Colok kamera via OTG. Buka menu pengaturan (titik tiga) > Stream Settings.
    • Ubah Type menjadi RTMP/RTSP Server.
    • Masukkan URL dan Stream ID. Centang Background Audio. Tekan START STREAM.

4️⃣ Monitoring (Preview Video)

  • Setelah HP melakukan live, tunggu sekitar 3 detik, lalu klik tombol 🔄 Start Preview.
  • Grafik garis di bawah layar akan mulai melompat sesuai dengan kestabilan sinyal di lapangan, dan video akan muncul di layar utama.

5️⃣ Integrasi ke OBS Studio

  • Buka OBS Studio. Tambahkan Source baru berjenis Media Source.
  • Hilangkan centang pada opsi Local File.
  • Di kolom Input, paste link dari Media Source (SRT).
  • Di kolom Input Format (tepat di bawahnya), ketik: mpegts
  • Klik OK. Video mentah HEVC kualitas maksimal dari HP akan langsung tersedot ke dalam OBS Anda!

💡 Tips & Troubleshooting

  • Layar Preview Blank Hitam? Tekan tombol ⏹️ Stop, tunggu 2 detik, lalu tekan 🔄 Start Preview kembali.
  • Delay Tinggi di OBS? Buka properties Media Source di OBS, turunkan Network Buffering menjadi 1MB atau 2MB.
  • Lupa Menyalakan VPN? Matikan Core Engine, nyalakan Tailscale, lalu nyalakan Core Engine lagi.
