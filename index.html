📖 BUKU PANDUAN PENGGUNAAN IRL SYSTEM V1

1️⃣ Pemilihan Jalur Jaringan (IP Source)

  • Tailscale VPN: Untuk produksi jarak jauh/beda lokasi kota. HP & Laptop wajib Connect Tailscale.
  • Jaringan Lokal: Gunakan Router Wi-Fi bawaan sendiri (tanpa internet tetap jalan) untuk latency paling minim.
  • Mobile Hotspot: Nyalakan Hotspot Windows laptop, lalu sambungkan Wi-Fi HP langsung ke laptop.

2️⃣ Panduan Setting Aplikasi HP Lapangan (Encoder)

  • Jalur Utama (SRT): Selalu utamakan protokol SRT karena jauh lebih instan dan anti-delay dibanding RTMP.
  • Aplikasi IRL Pro / Larix Broadcaster (Android/iOS):
    • Masuk Settings > SRT Latency: Ubah ke 120 msec atau 200 msec.
    • Keyframe Interval: Kunci di 1 second (30 frames).
    • B-Frames / Frame Reordering: Wajib OFF / 0.
  • Aplikasi Moblin (Khusus iOS / iPhone):
    • Aplikasi terbaik dan paling stabil untuk SRT di iPhone.
    • Masuk ke pengaturan Stream, pilih opsi Low Latency Mode.
    • Masukkan URL SRT PUSH dan pastikan kolom StreamID terisi kode publish secara presisi (misal: #!::r=irl1,m=publish).
  • Kamera Eksternal via USB Camera Pro (Android + Capture Card):
    • Gunakan mode ini jika HP disambungkan ke Handycam/Camcorder/DSLR lewat HDMI to USB Capture Card.
    • Di aplikasi USB Camera Pro, buka menu Live Streaming Settings.
    • Pilih jenis protokol (SRT/RTMP), atur resolusi tangkapan UVC ke 720p/1080p, dan pastikan fitur Network Buffer di-disable agar tidak menumpuk delay di HP.

3️⃣ Tarik Sinyal ke OBS Studio (REAL-TIME NO DELAY)

  • Tambah Media Source baru di OBS, lalu hilangkan centang pada Local File.
  • Masukkan link PULL SRT ke kolom Input. Tambahkan &latency=0 atau &latency=50 di paling akhir link.
  • KUNCI UTAMA: Ubah angka Network Buffering menjadi 0 MB.
  • Centang Use hardware decoding agar VGA/GPU laptop yang bekerja mengolah video.

4️⃣ Menampilkan Layar Preview Monitor

  • Klik 🟢 Start Core Engine lalu 🔄 START PREVIEWS.
  • Layar monitor di aplikasi ini sengaja dibuat dalam versi ringan (360p @20fps) agar CPU laptop tidak jebol. Tarikan di OBS Studio akan tetap masuk murni Full HD!

5️⃣ Fitur DVR Record & Auto-Save

  • Tekan ikon roda gigi ⚙️ di samping tombol DVR untuk mengatur Resolusi, FPS, dan Bitrate rekaman.
  • Klik PILIH FOLDER untuk menentukan lokasi folder asli di Windows kamu.
  • Nyalakan DVR AUTO-RECORD untuk otomatis merekam semua kamera saat sinyal memancar dari HP.
